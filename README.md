## 📌 Dynamic Form Builder (Laravel)

### 📖 Description

A Dynamic Form Builder built with Laravel that allows users to create, manage, and customize forms easily. It supports multiple field types, validation, and data storage. Users can generate forms dynamically and export submissions for real-world use.

---

### 🚀 Features

* Create dynamic forms without coding
* Multiple input fields (text, email, number, etc.)
* Form validation
* Store and manage form submissions
* User-friendly interface
* Export data (PDF/CSV if added)

---

### 🛠️ Technologies Used

* PHP
* Laravel
* MySQL
* Blade Template Engine
* HTML, CSS, JavaScript

---

### ⚙️ Installation

```bash
git clone https://github.com/your-username/dynamic-form-builder.git
cd dynamic-form-builder
composer install
cp .env.example .env
php artisan key:generate
```

👉 Configure database in `.env` file

```bash
php artisan migrate
php artisan serve
```

---

### 📂 Project Structure

* routes/ → Application routes
* app/Models → Database models
* app/Http/Controllers → Controllers logic
* resources/views → Blade UI files
